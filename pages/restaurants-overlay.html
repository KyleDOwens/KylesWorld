Guide

This page holds all the restaurants in San Antonio that I have been to, or want to go to!

There is a Map View which shows the physical locations of each restaurant, and a Table View which lists out each restaurant individually.

Each marker on the map represents a restaurant! Clicking a marker will bring up more information about the restaurant, including a link to it on Google! Places I have visited are colored green, with the shade determining how much I enjoyed it: low -> meidum -> high. Places I want to go are colored blue, with the shade determining how badly I want to go: low -> meidum -> high.

Select filters to limit which restaurants are shown! You can add a Distance Filter to only show restaurants that are within a certain distance (of time) from a starting point (fun fact, the map drawn by doing this is called an isochrone). To do this, (1) enter in the the max travel time in minutes (2) click the Add Isochrone button (3) click on the map at your starting location.

Clicking the Share Selections button will create a custom URL that leads to your exact map! It will be automatically copied to your clipboard.
